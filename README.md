# Microservice with Database Abstraction

A Python microservice that supports multiple databases (SQL Server, MySQL, PostgreSQL, Oracle) with easy switching capability.

## Features

- FastAPI web framework
- SQLAlchemy ORM with database abstraction
- Repository and Service pattern
- Docker support for all databases
- Configuration-based database switching
- Connection pooling
- Alembic migrations support
- RESTful API endpoints

## Supported Databases

1. **SQL Server**: Using pyodbc
2. **MySQL**: Using mysqlclient/pymysql
3. **PostgreSQL**: Using psycopg2/asyncpg
4. **Oracle**: Using cx_Oracle
